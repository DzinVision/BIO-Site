Značilnosti

Za nepopolno preobrazbo je značilno, da se iz jajčec izležejo ličinke, ki z rastjo in razvojem postajajo vse bolj podobne odraslim živalim. Skupine žuželk, za katere je značilna nepopolna preobrazba so npr. strigalice, stenice, kačji pastirji in enodnevnice.

Sledijo opisi nekaterih skupin žuželk z nepopolno preobrazbo

Ravnokrilci

Z več kot 20.000 vrstami so razširjeni povsod po svetu, razen v najhladnejših predelih. Večina vrst ima dolga krila, vendar v nevarnosti ne odletijo, ampak odskočijo z močnimi zadnjimi nogami. Sprednji par kril je trd in usnjat. Imajo žvekalne obustne okončine in se raznovrstno hranijo. Črički, zelene in dolgoroge kobilice ter njihovi sorodniki so plenilci ali vsejedi, poljske kobilice ali saranče pa so izključno rastlinojedi. Vse vrste imajo nepopolno preobrazbo. Številni samci stridulirajo oziroma »prepevajo«, da privabijo samice.

Strigalice

So podolgovate žuželke rjavkaste barve. Na široki glavi so dolge, nitaste tipalnice, ki imajo lahko tudi po 50 členov, razmeroma velike sestavljene oči in naprej usmerjene obustne okončine, izoblikovane v enostavno grizalo. Zadnja krila so razmeroma velika, polkrožne oblike in zelo tanka. Ličinke so podobne odraslim živalim, vendar nimajo kril in genitalij. Imajo tudi krajše tipalnice in razmeroma manjše klešče. So nočne živali. Podnevi poiščejo ozke špranje za počitek. Ponoči prilezejo iz svojih skrivališč in se pričnejo prehranjevati. So vsejedi mrhovinarji. Večina vrst redko leti, verjetno tudi zaradi težavnega zlaganja kril. Gibljive klešče na zadku uporabljajo v različne namene. Kadar se počutijo ogrožene, se postavijo v značilno držo z dvignjenimi kleščami. Včasih z njimi ujamejo plen, največkrat pa jih uporabljajo samci pri medsebojnih bojih in pri parjenju.

Bogomolke

Bogomolke so precej velike žuželke, ki so dobile ime po značilni drži prvega para nog, saj ju bogomolka pogosto drži predseboj kot v molitvi. Sprednji nogi sta posebej prilagojeni za grabljenje in jima pravimo grabilni nogi, ostala dva para nog sta hodilna. Vse bogomolke so izjemne plenilke. Njihov plen so zlasti žuželke, pa tudi predstavniki drugih skupin členonožcev, lotijo pa se celo drobnih vretenčarjev. Med parjenjem bogomolk samica svojemu partnerju včasih že med kopulacijo odgrizne glavo in ga kasneje požre.Odrasle bogomolke letijo, ko so v mirovanju imajo krila plosko zložena prek celotne dolžine zadka. Sprednji par kril pokriva večja zadnja krila, ki so pahljačaste oblike.
